#  Beach SA Hotel Management System

Sistema web para la gestión de hoteles, desarrollado con una arquitectura separada entre frontend y backend.

---

##  Descripción

Este proyecto implementa un sistema completo para la administración de un hotel, incluyendo gestión de clientes, reservaciones, usuarios y paquetes.

El sistema está dividido en dos partes principales:

-  Backend: API REST en .NET  
-  Frontend: Aplicación web en ASP.NET MVC  

---

##  Arquitectura

Usuario → Web App (MVC) → API → Base de Datos  

---

##  Funcionalidades

- Gestión de clientes  
- Gestión de reservaciones  
- Gestión de usuarios  
- Gestión de paquetes  
- Validaciones de datos  
- Consumo de API desde frontend  

---

##  Tecnologías

- C# / .NET  
- ASP.NET MVC  
- Entity Framework  
- SQL  
- HTML, CSS, JavaScript  

---

##  Cómo ejecutar el proyecto

Clonar el repositorio:

```bash
git clone https://github.com/KristinOrozcoH/beach-sa-hotel-system.git
```


Luego:

- Abrir la solución en Visual Studio
- Configurar la base de datos con el script incluido (ScriptBD.sql)
- Ejecutar primero la API y luego la aplicación web
